# Code of Conduct

## Our Pledge

我们承诺为所有参与者提供开放、友好、尊重的社区环境，不因年龄、体型、可见或不可见残疾、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向而区别对待。

## Our Standards

鼓励的行为：

- 使用友好、包容的语言。
- 尊重不同观点和经验。
- 接受建设性反馈。
- 关注对社区和项目最有益的结果。
- 对其他社区成员表现同理心。

不可接受的行为：

- 性化语言或图像，以及不受欢迎的性关注或性骚扰。
- 挑衅、侮辱、贬损评论，以及人身或政治攻击。
- 公开或私下骚扰。
- 未经明确许可发布他人私人信息，例如住址、邮箱或私人联系方式。
- 其他在专业环境中不适当的行为。

## Enforcement Responsibilities

项目维护者负责解释和执行行为准则，并可删除、编辑或拒绝不符合行为准则的评论、提交、代码、Issue、Pull Request 或其他贡献。

维护者也可以对不适当、威胁性、冒犯性或有害的行为采取临时或永久限制参与的措施。

## Scope

本行为准则适用于项目仓库、Issue、Pull Request、讨论区，以及代表项目参与公共交流的场景。

代表项目的场景包括：使用官方项目邮箱、通过官方社交媒体账号发言，或在线下/线上活动中以项目代表身份出现。

## Reporting

如果你遇到或发现不可接受的行为，请通过仓库维护者公开的 GitHub 联系方式联系维护者。请尽量提供：

- 事件描述。
- 发生位置和时间。
- 涉及人员或链接。
- 相关截图或上下文。

维护者会尽量保护报告者隐私，并根据情况采取合适措施。

## Enforcement Guidelines

维护者会根据行为影响选择处理方式：

1. **Correction**：私下说明问题并要求停止相关行为。
2. **Warning**：正式警告，并说明继续违规的后果。
3. **Temporary Ban**：在一段时间内禁止参与项目互动。
4. **Permanent Ban**：永久禁止参与项目互动。

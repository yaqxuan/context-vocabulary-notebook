从语境开始

把真实片段，变成长期记忆。

只记录有助于回想的内容，再由间隔复习决定它何时回来。

01

记录当前语境

写下原句、目标单词和当前含义；场景重要时，可附加截图、音频或本地 MP4。

02

整理语境卡片

用标签、收藏和语言对组织内容。AI 建议是可选辅助，不会代替你对语境的理解。

03

使用 FSRS 复习

选择 Again 或 Good，重新查看句子和媒体，并通过本地统计观察学习进展。

真实应用界面

为电脑端专注学习而设计。

下方全部来自当前应用，不展示虚构的云端功能或在线服务。

你的单词属于你

这是你电脑上的单词本，不是别人的云服务。

核心词汇数据保存在本地 SQLite 数据库，媒体保存在本地 uploads 目录；何时导出和备份由你决定。

  • 核心功能无需注册账号。
  • OCR 和语音识别可以使用可选本地工具。
  • 网络 AI 功能只向你配置的服务商发送请求的文本或媒体；片段云端回退默认关闭。
  • 本介绍页不含分析脚本或第三方跟踪器。

在本机运行

从 GitHub 一键安装。

先进入准备安装的空目录，再执行适合系统的命令；项目会直接安装到当前目录。完成后打开终端中显示的本地地址。

Linux / macOS / WSL

curl --fail --location --retry 5 \
  https://raw.githubusercontent.com/yaqxuan/context-vocabulary-notebook/main/scripts/install.sh \
  | bash

Windows PowerShell

irm https://raw.githubusercontent.com/yaqxuan/context-vocabulary-notebook/main/scripts/install.ps1 | iex

可选的 FFmpeg、Tesseract、whisper.cpp、AI 配置、更新与备份说明，请查看中文用户手册。

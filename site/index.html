A context-first workflow

From a real moment to a lasting memory.

Capture only what helps you remember, then let spaced repetition decide when it should return.

01

Capture the context

Write the original sentence, target word, and meaning. Attach a screenshot, audio clip, or local MP4 when the scene matters.

02

Organize the card

Use tags, favorites, and language pairs. Optional AI suggestions help, but never replace your own context.

03

Review with FSRS

Choose Again or Good, revisit the sentence and media, and follow progress through local learning statistics.

The real application

Built for focused desktop study.

These screenshots come from the current application—no fictional cloud features or mock online service.

Your words stay yours

A notebook on your machine, not someone else’s cloud.

Core vocabulary data lives in a local SQLite database and media stays in a local uploads directory. Export and back up both on your schedule.

  • No account is required for core features.
  • OCR and speech recognition can run through optional local tools.
  • Network AI features send requested text or media only to the provider you configure; clip fallback is off by default.
  • No analytics or third-party trackers are used on this website.

Run it locally

Install from GitHub.

Open an empty destination directory, run the installer there, then open the local address printed in the terminal. The project is installed directly into that directory.

Linux / macOS / WSL

curl --fail --location --retry 5 \
  https://raw.githubusercontent.com/yaqxuan/context-vocabulary-notebook/main/scripts/install.sh \
  | bash

Windows PowerShell

irm https://raw.githubusercontent.com/yaqxuan/context-vocabulary-notebook/main/scripts/install.ps1 | iex

Optional FFmpeg, Tesseract, whisper.cpp, AI configuration, updates, and backup instructions are covered in the English user guide.
